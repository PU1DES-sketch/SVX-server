# PU1DES Network Control

Painel de controle para SvxReflector/SvxLink.

## Funções previstas

- Dashboard de repetidoras online
- Login administrativo
- Cadastro e remoção de repetidoras
- Kick/Ban/Unban
- Proteção contra TOT e TX travado
- Last Heard
- Logs e histórico
- Backup e reinstalação automática
